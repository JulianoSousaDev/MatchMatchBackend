# MatchMatch Backend

Backend para o aplicativo MatchMatch desenvolvido em React Native.

## Tecnologias

- Node.js
- NestJS
- TypeScript

## Instalação

```bash
# Instalar dependências
npm install

# Desenvolvimento
npm run start:dev

# Produção
npm run build
npm run start:prod
```

## Rotas Disponíveis

- `GET /` - Retorna mensagem de boas-vindas
- `GET /teste` - Rota de teste para verificar funcionamento do backend
